# Virtusa Mandatory Practice – Use Cases

## Overview

This repository contains three use case implementations using SQL, Java, and Python.
Each use case demonstrates core programming concepts and problem-solving skills.

---

## Technologies Used

* MySQL (SQL)
* Core Java
* Python 3

---

## Use Cases

### SQL – Digital Library Audit

* Designed a relational database system
* Implemented overdue book detection
* Performed category-wise analysis

---

### Java – Password Validator

* Validates password strength based on rules
* Uses loops and character methods
* Provides feedback and retry mechanism

---

### Python – FareCalc Travel Optimizer

* Calculates ride fare based on distance and vehicle type
* Applies surge pricing during peak hours
* Handles invalid inputs
