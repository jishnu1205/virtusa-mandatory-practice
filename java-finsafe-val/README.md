# Password Validator (Java)

## Overview

This project is a simple Java program that validates a password based on given rules.
It ensures the password is strong and provides feedback if it is invalid.

---


## Features

* Minimum 8 characters
* At least one uppercase letter
* At least one digit
* Displays specific error messages
* Repeats input until valid password is entered
